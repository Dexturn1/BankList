# 🏦 BankList

BankList is a simple web application that displays a list of banks with relevant details. It is designed to demonstrate frontend development skills, clean UI structure, and basic data handling.

---

## 🚀 Features

- 📋 Display a list of banks
- 🔍 Search and filter functionality (if implemented)
- 📱 Responsive design for different screen sizes
- ⚡ Fast and lightweight (static frontend)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
